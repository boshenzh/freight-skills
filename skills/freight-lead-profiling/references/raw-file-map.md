# Raw File Map / 原始资料引用指南

Local raw files:
`$HOME/.openclaw/workspace/shipping-rate-automation/raw/source-files/`

WeCom raw registry DocID + sheet_id: resolve from `$HOME/.openclaw/workspace/shipping-rate-automation/wecom/links.md` (created by `freight-onboard` skill on first install). Each company has its own.

## Files

### 运价表.xlsx
Use for route/price/service facts. Extract cautiously; rates may be time-sensitive.
Useful for: 航线, POD, 船司, 20GP/40GP/40HQ, POL, remarks, free days, overweight fees.

### 运价信息.docx
Use for unstructured route and service notes. Good for identifying currently promoted routes and wording.

### 每日运价简报.docx
Use as style/reference for summarizing route options and schedules.

### 推广模板.docx
Use as style reference for concise customer-facing promotion. Do not copy blindly.

### 跟进话术.docx
Use for tone and follow-up framing. Rewrite based on actual customer context.

### 报价模板_报价跟进表_客户需求登记.xlsx
Use for future quotation/follow-up structure; not needed for initial lead profiling unless asked.

### 服务应用方案.docx
Historical requirement reference. Latest scenario rules override it.

### 场景变更.pdf
Latest scenario-change source. Current Scenario 1 v2 is based on this.
