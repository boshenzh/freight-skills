# Follow-up Templates / 跟进话术模板

These are distilled from the raw 跟进话术 document. They are reusable patterns for later stages, not the first lead-profiling draft unless relevant.

## 1. After sending rate / same day

```
刚刚把{航线/目的港}的运价和船期发您了，目前这几条是市场上综合来看比较有优势的方案。如果您对时效、免柜期、货重或目标价有特殊要求，可以发我，我再帮您筛一版更贴合的。
```

## 2. No reply after half day / next day

```
昨天给您发的{POL-POD}方案还有其他问题吗？如果当前船期赶不上，我可以补发后续一水；如果主要是价格问题，也可以告诉我目标价，我看看是否有更适合的舱位或能否再申请。
```

## 3. Customer hesitates on price

```
这个方案主要优势在{直航/免柜期/航程/费用包含项}，如果把目的港费用、免柜期和时效一起算，综合成本会更稳一些。当然如果您有预算或目标价，也可以发我，我帮您再争取或对比其他船司方案。
```

## 4. Before weekend / before closing date

```
提醒您一下，{目的港/航线}这条船期预计{截关时间}，如果这批货近期能好，刚好可以赶上这一水。否则可能要顺延到下一水，价格和舱位都有变化风险。需要我先帮您做预配/占舱确认吗？
```

## 5. Final gentle close

```
这批{目的港/航线}舱位预计{时间}前要确认。如果您近期没有出货计划，我这边就先不预留舱位了。后续有新船期或更合适价格，我再第一时间发您参考；您有其他目的港需求也随时叫我。
```

## Follow-up rules

- Mention concrete sailing/closing date only when known.
- Mention target price only as a question; do not claim we can always meet it.
- Adapt to customer's exact route and previous response.
- Avoid repeated pressure; switch to “后续有新船期再同步” after final follow-up.
