# Outreach Templates / 开发信与拓客模板

Use these as patterns, not copy-paste scripts. Apply `../high-reply-development-letter-playbook.md` before drafting. Always personalize with evidence from the customer's website and keep uncertain logistics details as questions.

## First-touch development email / message

### Structure

0. Short subject line tied to company/product/logistics value
1. Greeting + specific customer evidence
2. Why this may matter logistically
3. One relevant freight value only
4. One low-friction next action

### Chinese template

```
主题：{公司名}{产品/区域}物流方案参考

{联系人或称呼}您好，

看到贵司主要做{主营产品/业务}，官网上也提到{公开证据：海外市场/OEM/项目/产品线等}。这类产品如果有{目标区域或海外订单，若不确定写“海外/区域市场”}方向的出货，通常会比较关注船期稳定、综合成本和目的港费用透明度。

我们这边主要协助客户做海运方案对比和订舱安排，尤其可以按{可能相关航线/区域，如中东、红海、印巴、美线；不确定则写“目标市场”}整理不同船司的价格、截关时间、航程、免柜期和附加费差异，方便您评估整体成本。

如果贵司近期有海外出货计划，我可以先按您的目标市场做一版参考船期/运价对比。方便的话，您可以发我：目的港、起运港、箱型/柜量、货好时间和大概货重，我来帮您初步看方案。
```

### English template

```
Subject: {Company} - shipping options for {product/market}

Hi {Name},

I noticed that {Company} focuses on {products/business}, and your website mentions {public evidence}. For this type of cargo, freight planning often depends on sailing stability, total landed cost, free time, and transparent destination/local charges.

We help exporters compare ocean freight options across carriers, including rate level, closing date, transit time, free time, and key surcharges. If you are shipping to {target region if evidenced; otherwise "overseas markets"}, we can prepare a quick reference comparison for your team.

If useful, please share the destination port, POL, container type/volume, cargo ready date, and approximate cargo weight. I can help check suitable sailing/rate options for your review.
```

## When customer business fit is unclear

```
{称呼}您好，看到贵司主要做{主营业务}。我这边是做国际海运/出口物流方案的，想先确认一下贵司目前是否有海外整柜或拼箱出货需求？

如果有，我可以按目的市场帮您整理近期船期、价格和注意事项；如果暂时没有，也不打扰，后续有出口计划可以随时交流。
```

## When route is unknown

```
目前我还不能从公开信息确认贵司主要出口市场。为了避免给您发不相关的方案，想先确认一下：贵司近期主要看哪些目的国/目的港？我可以按对应航线整理船期和运价参考。
```

## When information is insufficient

Do not produce a confident pitch. Use:

```
公开信息暂时不足，建议先补充官网/主营产品/目标市场/联系人后再生成开发信。
```
