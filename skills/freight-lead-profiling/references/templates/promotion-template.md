# Promotion Template / 运价推广模板

Use after extracting actual rates from 运价表 or 运价信息. Do not invent rates.

## Compact route promotion format

```
{起运港/区域}近期{航线/区域}优势参考：

{POL}-{POD1} {20GP/40HQ or 20/40} {截关/船期} {直航/中转/航程/免柜期亮点}
{POL}-{POD2} {20GP/40HQ or 20/40} {截关/船期} {直航/中转/航程/免柜期亮点}
{POL}-{POD3} {20GP/40HQ or 20/40} {截关/船期} {直航/中转/航程/免柜期亮点}

以上价格/船期以订舱确认为准。若您有具体目的港、箱型、货重和货好时间，我可以帮您再筛一版更贴合的方案。
```

## Notes

- Only include routes and prices present in current source files or user-provided data.
- Prefer concise fields: POL, POD, price, closing date/sailing, direct/transshipment, key surcharge/free-time note.
- If a source says “实单单询”, do not quote a fixed rate; say “实单确认”.

## Lead outreach integration rule

When drafting 待审核开发信, do not stop at generic “we can compare rates” wording if current route/rate facts exist in 运价表.xlsx or 运价信息.docx.

Add a short factual route block only when the route/rate source matches the customer evidence:
- customer website/public info shows target markets covered by the rate file, e.g. Middle East / Red Sea / Gulf / Africa for current SK/NS Red Sea rate sheets;
- customer's products plausibly fit the service constraints, with no unsupported dangerous/sensitive-cargo promise;
- origin also plausibly fits the source POL/region, e.g. Shenzhen/Guangdong exporters for SK/NS/Guangzhou rates.

Do not paste the same promotion block into every lead. If the source rate file only contains Red Sea routes and the customer evidence points mainly to US/Europe/global-without-region, omit the rate block and ask for target destination first.

Keep the route block factual and compact:
- only use POL/POD, USD rate, CLS/cutoff/sailing, direct/transshipment and key free-time/surcharge notes that appear in source files;
- no vessel name required;
- if the route is not confirmed as relevant to the customer, introduce it as “如果红海/中东方向有出货，可参考…” rather than implying demand;
- if no destination is known, use it as a sample promotion block plus CTA asking for destination port, box type, cargo ready date;
- never invent rates, ports, free time, surcharges, or validity.
