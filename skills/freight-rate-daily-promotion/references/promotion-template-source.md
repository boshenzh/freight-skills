# Promotion Template Source / 推广模板原文（客户外发版）

> 这是发给客户的「推广信息」模板。从《每日运价简报》提取后**重新整理**：仅保留 POL / 运价 / 截关 / 直航或中转标识；**禁止**包含船名、成本价、底价、内部备注。

## 规则

1. 一条 POD 一行：`POL-POD USD<20GP>/<40GP> CLS<日期> <直航/中转>`
2. 默认**不写船名**。
3. **不写成本价、底价、采购价、利润、内部备注**。
4. 同 POD 多船司只放最优一条（最低 20GP，或客户偏好的直航优先）。
5. 单询占位允许：`SK-HODEIDAH 实单单询 JIB`。
6. 块末固定加一行有效性说明 + 询价邀约。

## 模板

```
{航线/区域}近期优势参考（更新：{YYYY-MM-DD}）

{POL}-{POD1} USD{20GP}/{40GP} CLS{截关日期} {直航/中转}
{POL}-{POD2} USD{20GP}/{40GP} CLS{截关日期} {直航/中转}
{POL}-{POD3} USD{20GP}/{40GP} CLS{截关日期} {直航/中转}
{POL}-{POD4} 实单单询 {中转港}

以上均为直航/中转服务，价格已含已确认基础附加费，具体以订舱确认为准。
欢迎询价订舱，更多船期及中转方案请随时联系！
```

## 提取规则（从内部简报 → 推广外发）

| 内部简报 | 推广外发 |
|---------|---------|
| 船司分块标题（"PIL 红海"） | 合并到区域块标题（"红海航线近期优势参考"），不显式带船司 |
| 同 POD 多船司报价 | 仅保留最优（最低 20GP，或客户偏好直航优先）|
| 船名 / 航次号 | ❌ 删除 |
| 中转天数 / 免柜期 / 附加费 | 仅在客户关键时保留（如"NSA 含 CIC USD150/300"）|
| 操作变更 / 跳港 | ❌ 删除 |
| 成本价 / 底价 / 内部备注 | ❌ 严格删除 |
| 单询占位 | ✅ 保留 |
